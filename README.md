# SQL Sales Project

Bu projede basit bir satış veritabanı oluşturdum.

SQLite kullanarak customers, products ve sales tablolarını oluşturdum. 
Python üzerinden bu veritabanına bağlanarak veri ekledim ve SQL sorguları çalıştırdım.

JOIN kullanarak satış tablosu ile ürün tablosunu birleştirip toplam ciroyu hesapladım.

Bu projede denediğim konular:

- SQL tablo oluşturma
- INSERT işlemleri
- JOIN kullanımı
- SUM fonksiyonu
- Python ile SQLite bağlantısı

Amaç, SQL ve Python birlikte nasıl çalışır onu görmekti.
